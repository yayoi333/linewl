コピペさんとタグ付けさん

LINE Creators Market申請用のブックマークレット登録ページです。スマホでは分割版2本の登録がおすすめです。

まずはドラッグ登録

PCでは、下の緑のリンクをブックマークバーへドラッグアンドドロップすると登録できます。

ドラッグ登録ができない場合やスマホで登録する場合は、「コードをコピー」して、下の手動登録の説明に沿ってブックマークのURL欄へ貼り付けてください。

分割版

スマホで動かしやすいよう、処理を2本に分けています。各リンクをブックマーク登録してください。

1タイトル説明コピペさん

切り出しくん・絵文字切り出しくんではタイトル、説明、絵文字のメイン画像番号をコピーし、LINE申請画面では英日欄やメイン画像欄へ貼り付けます。スマホ用の手動コピー欄も出ます。

タイトル説明コピペさん

2タグ付けさん

LINEのタグ付け申請画面で、タグJSONを貼り付けて自動実行します。貼り付け欄にはクリップボード内容を先に入れ、余計な前後テキストは無視します。

タグ付けさん

スマホ・手動登録

  1. このページをいったん普通にブックマークします。
  2. 作ったブックマークを編集します。
  3. 名前を タイトル説明コピペさん または タグ付けさん にします。
  4. URL欄を削除して、上でコピーした javascript:... を貼り付けます。
  5. 取得元ページ、またはLINE申請画面で、そのブックマークを開くと実行されます。

コピーできない場合は、各欄を全選択してURL欄に貼り付けてください。先頭の javascript: を消さないでください。

スマホで実行するコツ

  1. 対象ページを開いた状態で、アドレスバーにブックマーク名を入力します。
  2. 検索候補ではなく、登録したブックマークを選びます。
  3. タイトル説明コピペさんでは、取得元ページでデータ欄が出たら、コピーできていない時だけ手動コピーします。
  4. タグ付けさんでは、LINE側で貼り付け欄が出たら、タグJSONを確認して実行します。

URL欄が検索結果になったり、javascript: が消えている場合は実行されていません。iPhoneではSafariが比較的安定します。

🦊 Firefox Android ＋ Violentmonkey 版（Opera で動かない場合）

Opera Android ではブックマークレットが動作しない場合があります。Firefox Android と Violentmonkey 拡張を使うと、LINEのタグ付けページに緑のボタンが自動で表示されます。ブックマークレットより安定して動作します。

  1. Firefox for Android をインストールします。
  2. Firefox を開き、Violentmonkey 拡張をインストールします（Firefox Add-ons で検索）。
  3. 下の「スクリプトをコピー」を押してコードをコピーします。
  4. Violentmonkey のアイコンをタップ → ＋ 新しいスクリプトを作成 を押します。
  5. 内容を全部削除してからコピーしたコードを貼り付け、保存します。
  6. LINE Creator Market のタグ付け申請ページを開くと、画面右下に 🏷 タグ付けさん ボタンが表示されます。

対応しやすいブラウザ

  • PC: Chrome, Edge, Firefox, Safari, Brave, Vivaldi
